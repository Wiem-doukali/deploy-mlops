# Medical Diagnosis MLOps API

API de diagnostic médical avec reconnaissance vocale, déployée avec Azure MLOps.

## Architecture

- **FastAPI** pour l'API REST
- **Scikit-learn** pour les modèles ML
- **Docker** pour la conteneurisation
- **Azure Container Apps** pour le déploiement
- **GitHub Actions** pour CI/CD
- **MLflow** pour le tracking des modèles
